
# 🤖 Yapay Zeka Asistanım (AI Chatbot)

Bu proje, tarayıcı üzerinden doğrudan Google Gemini API'sini kullanarak çalışan, şık ve minimalist tasarıma sahip bir yapay zeka sohbet sitesidir. GitHub Pages üzerinde tamamen ücretsiz ve sunucusuz (serverless) olarak barındırılmaktadır.

## 🚀 Özellikler
* **Canlı Yapay Zeka Deneyimi:** Google'ın güçlü `gemini-1.5-flash` modelini kullanır.
* **Modern ve Duyarlı Arayüz:** Hem bilgisayarlardan hem de mobil cihazlardan rahatça kullanılabilir.
* **Hızlı Kurulum:** Sadece tek bir `index.html` dosyası ile çalışır, karmaşık kurulumlar gerektirmez.

## 🛠️ Nasıl Çalıştırılır?

Projenin kendi bilgisayarınızda veya kendi GitHub sayfanızda çalışması için şu adımları izleyebilirsiniz:

1. Bu depoyu (repository) klonlayın veya `index.html` dosyasını indirin.
2. [Google AI Studio](https://aistudio.google.com/) adresinden ücretsiz bir **Gemini API Anahtarı (API Key)** alın.
3. `index.html` dosyasını bir kod editörüyle açın ve `BURAYA_KENDI_API_ANAHTARINI_YAZ` kısmına kendi anahtarınızı yapıştırın.
4. Dosyayı kaydedip tarayıcınızda açın veya GitHub Pages üzerinden yayına alın!

## 🧰 Kullanılan Teknolojiler
* HTML5
* CSS3 (Flexbox & Modern UI Tasarımı)
* JavaScript (Fetch API & Asenkron Programlama)
* Google Gemini API

## ⚠️ Güvenlik Notu
Bu proje eğitim ve kişisel kullanım amacıyla geliştirilmiştir. API anahtarınız `index.html` içinde açık kaynaklı olarak yer aldığı için, üretim (production) ortamlarında veya sınırlı/ücretli API anahtarlarıyla bu yöntemin kullanılması güvenlik riski oluşturabilir.
